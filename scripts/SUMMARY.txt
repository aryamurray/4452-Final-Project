================================================================================
SCRIPT DESCRIPTIONS
================================================================================

1. eval_clip_zeroshot.py
   Purpose: Zero-shot Chinese character recognition using CLIP
   Features:
     - Multilingual CLIP (Chinese prompts)
     - English CLIP (English prompts)
     - Top-1/5/10 accuracy metrics
     - Text embedding caching
   Runtime: ~15-45 minutes
   Output: clip_zeroshot_results.json

2. eval_structural.py
   Purpose: Evaluate structural constraints and radical filtering
   Features:
     - Alpha hyperparameter sweep [0.3, 0.5, 0.7, 0.9]
     - Tests on TinyCNN, TinyCNNJoint, MobileNetV3
     - Before/after accuracy comparison
     - Optimal alpha selection
   Runtime: ~20-30 minutes
   Output: structural_results.json

3. eval_bigram_settings.py
   Purpose: Test bigram language model effectiveness
   Features:
     - Setting 1: Random character pairs
     - Setting 2: Real word pairs (high-frequency)
     - Setting 3: Adversarial pairs (confusable chars)
     - Re-ranking impact analysis
   Runtime: ~15-25 minutes
   Output: bigram_settings_results.json

4. run_benchmarks.py
   Purpose: Comprehensive efficiency benchmarking
   Features:
     - Model size measurement
     - Inference latency profiling
     - Parameter counting
     - Quantization testing (int8)
     - Pareto frontier plotting
   Runtime: ~10-20 minutes
   Output: benchmark_results.json, benchmark_table.csv, efficiency_pareto.png

5. run_all.py
   Purpose: Orchestrate complete pipeline
   Features:
     - Build all resources
     - Train all models
     - Run all evaluations
     - Generate summary report
     - Conditional execution (skip flags)
   Runtime: ~8-12 hours (full), ~1-2 hours (eval only)
   Output: All results + summary_report.txt

================================================================================
QUICK START
================================================================================

Evaluate Everything (models already trained):
  $ python scripts/run_all.py --skip-build --skip-training

Run Individual Evaluations:
  $ python scripts/eval_clip_zeroshot.py
  $ python scripts/eval_structural.py
  $ python scripts/eval_bigram_settings.py
  $ python scripts/run_benchmarks.py

Full Pipeline from Scratch:
  $ python scripts/run_all.py

Common Issues:

  3. CLIP Download Fails → Check internet connection
  4. Slow Evaluation → Use cached indices

================================================================================
PERFORMANCE NOTES
================================================================================

Caching Strategy:
  - Dataset indices: Cached as .pkl files
  - CLIP text embeddings: Cached as .pt files
  - First run builds caches (~2x slower)
  - Subsequent runs use cache (much faster)

Memory Usage:
  - CLIP: 6-8 GB GPU
  - Other evals: 2-4 GB GPU
  - Can run on CPU (slower)

Parallelization:
  - Scripts are independent
  - Can run on multiple GPUs simultaneously
  - See EXAMPLES.md for parallel execution
